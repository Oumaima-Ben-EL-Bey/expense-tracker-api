# Expense Tracker API
[![CI](https://github.com/Oumaima-Ben-EL-Bey/expense-tracker-api/actions/workflows/ci.yml/badge.svg)](https://github.com/Oumaima-Ben-EL-Bey/expense-tracker-api/actions/workflows/ci.yml)

## Description 
A REST API for tracking expenses. Built as a portfolio project to learn the Spring Boot ecosystem end-to-end.
## Status
In active development. Application scaffolding and one trivial endpoint are in place; expense functionality is not yet implemented.    
## Tech Stack
- Java 21
- Spring Boot 4
- Maven
## Running Locally

Requires Java 21.

```bash
./mvnw spring-boot:run        # macOS/Linux
mvnw.cmd spring-boot:run      # Windows
```
## Endpoints
- `GET /hello` — returns a greeting message
- `GET /version` — returns the current application version
